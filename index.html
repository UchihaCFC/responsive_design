¿Qué es Responsive Design y por qué lo debo utilizar?

Con responsive design estaremos maquetando nuestros diseños de forma que se puedan visualizar desde cualquier dispositivo con internet en la actualidad, ya sea un ordenador de sobremesa, un netbook, una tablet (Ipad), un smartphone (Iphone) ... etc. Gracias a seguir las pautas del responsive design nuestro maquetado se adaptará sin ningún problema a todo tipo de resoluciones con una correcta visualización.

Para poder ver con mucha más claridad lo que significa que un diseño se adapte podemos descargar Windows Resizer desde el Chrome store. y hacer pruebas con la web que haremos con skeleton o antes de empezar con la web de Mejorando.la. Podremos ver que a medida que vamos cambiando de resolución con el plugin de Chrome, los divs se van autoajustando facilitando la visualización y lectura de los mismos.

Usar Frameworks para Responsive Design

Al igual que los backend's tienen frameworks para facilitarles algo más la vida a la hora de hacer su trabajo, los frontend's con el responsive design no nos podíamos quedar atrás. es por eso que disponemos de diversos frameworks que nos ayudaran en la tediosa tarea de crear un web adaptada a todos los navegadores del mercado y a la vez compatible con todos los dispositivos.

Hay ya muchos frameworks famosos como son Bootstrap , Iunit css o Get Skeleton que es el que usaremos para este ejemplo por su simplicidad para comenzar y asentar las bases de Resposive Design para más adelante poder entender con facilidad Bootstrap y que no se nos haga nada complicado.
Todo están basados en un sistema grid por columnas lo que nos ayudará a crear la estructura de nuestro diseño y organizarla de forma clara y ordenada desde un principio.
